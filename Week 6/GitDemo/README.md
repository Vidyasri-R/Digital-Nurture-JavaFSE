# GitDemo

A demonstration project for learning Git commands.

## Steps Covered

1. **Git Configuration** - Setting up user.name and user.email
2. **Editor Integration** - Integrating notepad++ as default editor
3. **Source Code Management** - git init, git add, git commit, git push, git pull

## Commands Used

```bash
git version
git config --global user.name "username"
git config --global user.email "username@cognizant.com"
git config --global --list
git config --global core.editor "notepad++.exe -multiInst -nosession"
git init GitDemo
git add welcome.txt
git commit
git push origin master
git pull origin master
```
